常用UI及交互效果组件

常用UI及交互效果组件，主要是网站一些常见的交互效果以及一些比较好的css实践。
主要包括图文混排、负边距应用、分隔线、选项卡select控件、单选及复选框、文本输入框、内容滚动、幻灯片、分页、弹出框等。本项目基于twitter-bootstrap架构。

注意:以上用到的一些交互效果都依赖于jquery库，所以在用的时候必须要引入jquery库，才能正常使用。

图文混排

说明

图文混排，最常见的了，一般用到的方法是，就是用浮动，用了浮动就必须定宽，结构就失去灵活性，同时必须解决浮动带来的一系列问题。下面推荐一种比较好的实践。

css代码片段

.item .pic { float:left;margin-right:10px; }

.item .content { overflow:hidden;zoom:1; } /* 或用display:table-cell */

不要忘记清除浮动哦，这个时候直接上clearfix啦！

        ....

html结构

        <input type="text" value="Enter Username here.." class="swap" />

js代码

        wap_val = [];
        $(".swap").each(function(i){
            wap_val[i] = $(this).val();
            $(this).focusin(function(){
                if ($(this).val() == swap_val[i]) {
                    $(this).val("");
                }
            }).focusout(function(){
                if ($.trim($(this).val()) == "") {
                    $(this).val(swap_val[i]);
                }
            });
        });

                  $(function(){  
                    var sw = 0;
                    $(".demo .num a").mouseover(function(){

                      myShow(sw);
                    });
                    function myShow(i){
                      $(".demo .num a").eq(i).addClass("cur").siblings("a").removeClass("cur");
                      $(".demo ul li").eq(i).stop(true,true).fadeIn(600).siblings("li").fadeOut(600);
                    }
                    //滑入停止动画，滑出开始动画
                    $(".demo").hover(function(){
                      if(myTime){
                         clearInterval(myTime);
                      }
                    },function(){
                      myTime = setInterval(function(){
                       myShow(sw)
                       sw++;
                       if(sw==3){sw=0;}
                      } , 3000);
                    });
                   //自动开始
                    var myTime = setInterval(function(){
                       myShow(sw)
                       sw++;
                       if(sw==3){sw=0;}
                    } , 3000);
                  })

分页

最常见的分页模式

一般的分页都是居中显示，并且分页的数字都是自适应的，因此在自适应的时候采用浮动的方式，而居中的方式就是采用一种叫float:center的思路，用相对定位后的div进行left:50%和left:-50%来居中显示。

html结构

        <div class="pager">
        <div class="page"><a href="javascript:void(0)">上一页</a> <a href="javascript:void(0)">1</a><a href="javascript:void(0)" class="current">2</a><a href="javascript:void(0)">3</a> <a href="javascript:void(0)">4</a><a href="javascript:void(0)">5</a><a href="javascript:void(0)">6</a><a href="javascript:void(0)">7</a><a href="javascript:void(0)">8</a> <a href="javascript:void(0)">9</a> <span>...</span> <a href="javascript:void(0)">99</a><a href="javascript:void(0)" class="dis_page">下一页</a></div>
</div>

css代码

        .pager {position:relative;float:right;right:50%;font:normal 12px/1.5em simsun, Verdana,Lucida, Arial, Helvetica, sans-serif;}
        .pager .page {position:relative;float:right;right:-50%;}
        .pager a {float:left;height:20px;overflow:hidden;margin:0 2px;padding:0 5px;line-height:20px;text-decoration:none;color:#000000;border:1px solid #CBCBCB;background-color:#E5E5E5;border-radius:2px;}
        .pager a.current,
        .pager a.current:hover,
        .pager a:hover {color:#EFEFEF;border:1px solid #000000;background-color:#242424;box-shadow:0 0 3px #000000;}
        .pager a.dis_page,
        .pager a.dis_page:hover {color:#CBCBCB;border:1px solid #CBCBCB;background-color:#E5E5E5;cursor:not-allowed;box-shadow:none;}
        .pager span {float:left;height:22px;overflow:hidden;margin:0 2px;line-height:28px;color:#242424;}
